# Contributing

Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.

## Code Quality
- Follow PSR-12 for PHP code.
- Write tests for new features.
- Document public APIs in OpenAPI.
- Emit domain events for state changes.
- Valid PhpStan 10 only please. We are currently refactoring old code for this. 

## Commit Messages
Use conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`, `test:`, etc.
                      (exceptions are okay every now and then, for funsies) - retroactive because i broke my own rule.
